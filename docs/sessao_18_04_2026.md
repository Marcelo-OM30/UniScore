## 1. Correção de bugs no fórum (`src/app/forum/page.js`)

### Bug 1 — Perda de foco ao digitar no textarea

**Causa raiz:** `CreatePostForm` e `PostCard` estavam declarados como `const` arrow functions *dentro* da função `Forum`. A cada tecla digitada, `setNewPost` atualizava o estado do `Forum`, fazendo o React recriar as definições dos componentes — tratando-os como novos tipos e desmontando/remontando o DOM, o que fazia o foco sair do textarea.

**Solução:** Mover `PostCard`, `CreatePostForm` e `formatTimestamp` para fora do `Forum` como funções de módulo (`function PostCard(...)`, `function CreatePostForm(...)` no topo do arquivo). O estado de cada formulário ficou local ao próprio componente.

### Bug 2 — Botão de comentários sem ação

**Causa raiz:** O botão de comentários não tinha `onClick` e só exibia a contagem.

**Solução:** Adicionado estado local `showComments` no `PostCard`. Ao clicar, expande uma seção inline com:
- Lista de comentários existentes (do array `comentarios` no Firestore)
- Campo de input para novo comentário
- Botão "Enviar" que salva via `arrayUnion` no documento do post

### Bug 3 — Botão de compartilhar sem ação

**Causa raiz:** O botão de compartilhar não tinha `onClick`.

**Solução inicial:** Implementado `handleShare` com:
- `navigator.share({ title, text, url })` em dispositivos móveis
- `navigator.clipboard.writeText(url)` como fallback no desktop

**Melhoria posterior (compartilhamento com imagem):** Atualizado para Web Share API Level 2:
1. Faz `fetch` da imagem do Firebase Storage
2. Converte para `File`
3. Usa `navigator.canShare({ files })` para verificar suporte
4. Compartilha com `navigator.share({ files: [file] })` se suportado
5. Fallback para texto + link caso contrário
